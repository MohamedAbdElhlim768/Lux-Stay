# Hotelio

A modern and responsive hotel website built using HTML, CSS, JavaScript, Bootstrap, and jQuery.

## Features
- Hero section with full-screen background
- About section
- Services section
- Projects/Portfolio showcase
- Smooth scrolling
- Interactive buttons and hover effects
- Responsive layout for mobile and desktop

## Technologies
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript
- Bootstrap 5
- jQuery 3.6
- Font Awesome for icons
